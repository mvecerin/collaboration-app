# Collaboration-app

MERN stack web app for group messaging and tasks using Socket.io and Redux Toolkit

### Run locally:

Create .env file:
```sh
TOKEN_SECRET="your_secret"
DB_URI="link_to_mongo_db"
NODE_ENV=development
```

Start the server:
```sh
npm run dev
```
Start the client (in client directory):
```sh
cd client
npm start
```
